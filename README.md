# SQL3
Different structures JOINs.

Описание:
Проект посвящён углублённому изучению соединений таблиц в SQL и принципов реляционной алгебры. В рамках заданий выполнялся анализ связанных данных о клиентах, пиццериях, заказах и посещениях с использованием различных типов JOIN и CTE.

## Этапы выполнения проекта:
- Практика LEFT, RIGHT и FULL JOIN для выявления пропусков и несвязанных данных.
- Поиск отсутствующих данных с использованием generate_series и временных интервалов.
- Рефакторинг запросов с применением CTE (Common Table Expressions).
- Построение запросов с объединением нескольких таблиц для анализа предпочтений клиентов и цен.
- Решение аналитических задач с фильтрацией, сортировкой и сравнением данных между сущностями.

## Освоенные компетенции:
- Глубокое понимание принципов работы JOIN в SQL и их отличий.
- Использование LEFT / RIGHT / FULL JOIN для анализа полноты данных.
- Применение CTE для повышения читаемости и поддержки сложных запросов.
- Навыки поиска пропусков и аномалий в данных.
- Построение сложных аналитических SQL-запросов на основе нескольких связанных таблиц.
- Понимание связи SQL-запросов с концепциями реляционной алгебры.
